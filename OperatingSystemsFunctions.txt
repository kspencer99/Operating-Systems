Microsoft Windows 10 provides virtual desktops. 
It is called "Task View" and you can access them on the Taskbar. 
To add a new desktop, you just click the plus sign.
You can create multiple desktops and when you want to switch between them, you just click the Task view button and move your mouse over the one you want.
Chrome OS uses applications that can be run from the cloud in the Chrome browser.
It is 100% web based.
There are no desktop applications.
Chrome OS is quick to start up.
